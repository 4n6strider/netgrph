# API Documentation

* The API is currently under development and in flux, coming soon.

### Networks

* /nets - list of all
* /nets?group=X
* /nets?cidr=X
* /nets?ip=X
* /nets?nlist=X
* /nets?nfilter=X


* /devs - list of all
* /devs/{device}/vlans
* /devs/{device}/neighbors
* /devs/{device}/nets
* /devs?query='*'
* /devs?group='*'
* /devs?full=1


* /vlans
* /vlans&vid=X
* /vlans&vname=X
* /vlans&group=X


* /path
* /spath
* /rpath
* /fpath
