# Napalm Support

## NetGrph 

### Supported data
* Routed Interfaces
* LLDP Neighbors
* Model, Version, Serial, Uptime

### Missing Data

#### L3
* VRF
* Virtual gateways (HSRP/VRRP)

#### L2
* L2 VLAN List
* VLAN Names
* STP priority configuration

#### Interfaces
* Port-channel membership (match mac address?)
* Trunked VLAN configuration (1-4096 etc)
* Native VLAN ID

## NetDB Support
* MAC Table
* ARP Table
* Interface Table
